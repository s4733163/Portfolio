# Personal Portfolio

A personal portfolio website built with React and Vite, showcasing projects, skills, work experience, and contact information.

## Live Site

[https://s4733163.github.io/Portfolio](https://s4733163.github.io/Portfolio)

## Tech Stack

- **Frontend:** React 19, React Router
- **UI:** MUI (Material UI), Bootstrap Icons, Font Awesome, React Icons
- **Build Tool:** Vite
- **Email:** EmailJS
- **Maps:** Google Maps (via @vis.gl/react-google-maps)
- **Deployment:** GitHub Pages

## Pages

- **Home** - Landing page with introduction and resume download
- **About** - Skills, work experience, and timeline
- **Projects** - Showcase of personal projects
- **Contact** - Contact form and Google Maps integration

## Featured Projects

### TableTap
A full-stack restaurant ordering system with QR-based menus and vendor dashboards. Built with server-side rendering (SSR) for efficiency, using Django, HTML, CSS, JavaScript, and SQL ORM.
- [Live Site](https://tabletapuq.onrender.com/tabletap/)

### SmartRecipe
An AI-powered recipe generator that transforms your available ingredients into personalized, nutritious recipes using LangChain and the Gemini API. Built with Django REST Framework, React, Vite, TailwindCSS, PostgreSQL, and Cloudinary.
- [Live Site](https://smart-recipe-kpak.vercel.app/)

### CompliGen
Generates clear, tailored compliance documents (Privacy Policies, Terms & Conditions, and more) in minutes using RAG and Google Vertex AI embeddings. Built with React, Vite, TailwindCSS, Django REST Framework, PostgreSQL, and Gemini API.
- [GitHub](https://github.com/s4733163/CompliGen)

### QuickNotes
A lightweight browser extension for instantly capturing, saving, and organizing notes from any webpage. Built with JavaScript, HTML, CSS, and the Chrome Extension API.
- [GitHub](https://github.com/s4733163/WebNotes)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
