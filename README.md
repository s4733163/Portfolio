# Personal Portfolio

A personal portfolio website built with React and Vite, showcasing projects, skills, work experience, and contact information.

## Live Site

[https://s4733163.github.io/Portfolio](https://s4733163.github.io/Portfolio)

## Tech Stack

- **Frontend:** React 19, React Router
- **UI:** MUI (Material UI), Bootstrap Icons, Font Awesome, React Icons
- **Build Tool:** Vite
- **Email:** EmailJS
- **Maps:** Google Maps (via @vis.gl/react-google-maps)
- **Deployment:** GitHub Pages

## Pages

- **Home** - Landing page with introduction and resume download
- **About** - Skills, work experience, and timeline
- **Projects** - Showcase of personal projects
- **Contact** - Contact form and Google Maps integration

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
